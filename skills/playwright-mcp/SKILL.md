Provides a standardized MCP interface for Playwright capabilities, allowing the agent to perform multi-step browser interactions.

Usage:
Use this for advanced web-based tasks like filling forms, navigating multi-page workflows, or extracting structured data from web apps.

Parameters:
- script: A high-level description of the browser task or a structured command sequence.
